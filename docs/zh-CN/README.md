### 这里都会有什么?
所有关于 [轻文社](https://www.qinglite.com/ "QingLite") 的 API 请求。

### 请求方式
| key      | value |
| :-----------: | :-----------: |
| 强制编码格式      | **UTF-8**       |
| 返回值格式   | **JSON**        |
| 请求发起地址   | `https://api.qinglite.com/`        |

### 需要注意
在前期阶段，为了方便跨设备请求，后端（API）开发相对简单。包括但不限于 Cookies 和 Tokens 以及 Csrf，后期会重新开发！
